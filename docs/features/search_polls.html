Search Polls

Feature: Search Polls
  As a user
  I want to search polls
  So that I can interact with the system meaningfully

  Background:
    Given I am logged in
    And I am on the polls page

  Scenario: Successfully search polls
    When I enter valid information to search polls
    And I submit the form
    Then I should see a confirmation message
    And the polls should be saved

  Scenario: search polls using maximum input values
    When I fill in all fields with their maximum allowed values
    And I submit the form
    Then the system should accept the input
    And the polls should be saved without error

  Scenario: Attempt to search polls with missing required fields
    When I submit the form with incomplete information
    Then I should see validation messages for required fields

  Scenario: Attempt to search polls with invalid input types
    When I provide invalid data formats
    And I submit the form
    Then the system should reject the input
    And I should remain on the polls page

  Scenario: Navigate away and return to resume search polls
    When I start to search polls
    And I leave the page before submitting
    And I return to the polls page
    Then my previous input should be restored if autosave is enabled
